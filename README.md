# Self Study Notes

A collection of personal notes and summaries covering various software engineering topics, tools, and frameworks.

## Topics Covered

### DevOps & Infrastructure
- **[CKA](./CKA.md)**: Notes on Kubernetes architecture, installation, and core components like the API server, etcd, scheduler, and controllers.
- **[Docker](./Docker.md)**: Basics of containerization, Docker commands, and images vs containers.
- **[Kubernetes](./Kubernetes.md)**: Fundamental concepts including Pods, Nodes, Clusters, and Services.
- **[Networking](./Networking.md)**: Kubernetes networking models, CNI (Container Network Interface), and underlay/overlay networks.
- **[IAM](./IAM.md)**: Concepts like OAuth, LDAP, SAML, and authentication flows.

### Programming Languages & Frameworks
- **[Go](./Go.md)**: Core language features including Goroutines and Channels.
- **[Go Gin](./Go-gin)**: Setup and usage of the Gin web framework for Go, including CompileDaemon and GORM.
- **[React Native](./React-Native.md)**: Getting started with Expo, building, and publishing mobile apps.
- **[GraphQL](./GraphQL.md)**: Introduction to GraphQL, queries, mutations, and solving over/under-fetching.

### System Design & Architecture
- **[System Design](./System-design.md)**: Distributed systems concepts (CAP theorem), SOLID principles, and design patterns.
- **[DDD](./DDD.md)**: Core principles of Domain Driven Design, bounded contexts, and architectures like Hexagonal and Onion.
- **[DBMS](./DBMS.md)**: Database internals, disk structure, B-trees, and indexing.
- **[Multithreading & Concurrency](./Multithreading-and-Concurrency.md)**: Processes vs threads, race conditions, deadlocks, and synchronization primitives.

### Testing
- **[Testing](./Testing.md)**: Methodologies (Unit, Integration, E2E), specific notes on testing in Go, and tools like Cypress and Mock Service Worker.
