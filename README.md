# Self Study Notes

A collection of personal notes and summaries covering various software engineering topics, tools, and frameworks.

## Topics Covered

### DevOps & Infrastructure
- **[CKA](./DevOps/CKA.md)**: Notes on Kubernetes architecture, installation, and core components like the API server, etcd, scheduler, and controllers.
- **[Docker](./DevOps/Docker.md)**: Basics of containerization, Docker commands, and images vs containers.
- **[Kubernetes](./DevOps/Kubernetes.md)**: Fundamental concepts including Pods, Nodes, Clusters, and Services.
- **[Networking](./DevOps/Networking.md)**: Kubernetes networking models, CNI (Container Network Interface), and underlay/overlay networks.
- **[IAM](./DevOps/IAM.md)**: Concepts like OAuth, LDAP, SAML, and authentication flows.

### Programming Languages & Frameworks
- **[Go](./Languages-Frameworks/Go.md)**: Core language features including Goroutines and Channels.
- **[Go Gin](./Languages-Frameworks/Go-gin.md)**: Setup and usage of the Gin web framework for Go, including CompileDaemon and GORM.
- **[React Native](./Languages-Frameworks/React-Native.md)**: Getting started with Expo, building, and publishing mobile apps.
- **[GraphQL](./Languages-Frameworks/GraphQL.md)**: Introduction to GraphQL, queries, mutations, and solving over/under-fetching.

### System Design & Architecture
- **[System Design](./CS-Fundamentals/System-design.md)**: Distributed systems concepts (CAP theorem), SOLID principles, and design patterns.
- **[DDD](./CS-Fundamentals/DDD.md)**: Core principles of Domain Driven Design, bounded contexts, and architectures like Hexagonal and Onion.
- **[DBMS](./CS-Fundamentals/DBMS.md)**: Database internals, disk structure, B-trees, and indexing.
- **[Multithreading & Concurrency](./CS-Fundamentals/Multithreading-and-Concurrency.md)**: Processes vs threads, race conditions, deadlocks, and synchronization primitives.

### Testing
- **[Testing](./CS-Fundamentals/Testing.md)**: Methodologies (Unit, Integration, E2E), specific notes on testing in Go, and tools like Cypress and Mock Service Worker.
